## How to Use
  To run the files from the repository,
  you need to:
  1. Install [Node.js](https://nodejs.org/en)
  2. Clone this repository:
      ```bash
      $ git clone https://github.com/vladyatsuk/dev-school-hw5.git
      ```
  3. In the project folder, run: 
      ```bash
      $ node <file_name>
      ```
